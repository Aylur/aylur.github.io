Description

This signal is emitted when user input is required. The input should be hidden when entered (e.g., for passwords).

This signal has to be matched with exaclty one supply_secret call.

Default handler:

The default handler is called before the handlers added via g_signal_connect().

Parameters

msg

Type: gchar*

The prompt to be shown to the user.

The data is owned by the caller of the function.
The value is a NUL terminated UTF-8 string.
